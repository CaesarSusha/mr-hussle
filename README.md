This project is an attempt to create an application that will make it much easier for me to manage my daily task-tracking routine.
At the moment, I manually perform all of the calculations required to track my progress, which is both time-consuming and inefficient.
I still want to monitor my progress and reward myself accordingly, but I have yet to find an existing app that meets my requirements. As a result, I decided to build my own.

This is a work in progress.
